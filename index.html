🔧 คู่มือตั้งค่า Firebase + Google Sheets

1
สร้างโปรเจกต์ Firebase (ฟรี)
ไปที่ console.firebase.google.com → Add Project → ตั้งชื่อ → ปิด Google Analytics → Create Project
เปิด Firebase Console →
2
เปิด Realtime Database
ในโปรเจกต์ → เมนูซ้าย "Build" → "Realtime Database" → "Create database" → เลือก Asia-Southeast1 → Start in test mode (ใช้งานฟรี 30 วัน แล้วค่อยตั้ง Rules)
3
คัดลอก Config มาใส่ในไฟล์นี้
Project Settings (⚙️) → General → "Your apps" → กด </> (Web) → ลงทะเบียน → คัดลอก firebaseConfig แล้วแทนที่ค่าใน FIREBASE_CONFIG ด้านบนของไฟล์ HTML นี้
const FIREBASE_CONFIG = { apiKey: “AIza…”, authDomain: “myproject.firebaseapp.com”, databaseURL: “https://myproject-default-rtdb.asia-southeast1…”, projectId: “myproject”, … };
4
สร้าง Google Sheet + Apps Script
1. เปิด Google Sheets ใหม่ → ตั้งชื่อ "บิงซู ยอดขาย"
2. Extensions → Apps Script → ลบโค้ดเดิม → วางโค้ดด้านล่าง → Save → Deploy → New Deployment → Web App → Execute as: Me → Who has access: Anyone → Deploy → คัดลอก URL
const SHEET_NAME = "ยอดขาย"; const HEADERS = [“วันที่”,“พนักงาน”,“M ขาย”,“L ขาย”,“บานาน่า”,“ครีมชีส”,“รวม”,“QR”,“เงินสด”,“Grab เต็ม”,“Grab สุทธิ”,“LineMAN เต็ม”,“LineMAN สุทธิ”,“Gross”,“สุทธิ”,“ช่วงต่ำ”,“ช่วงสูง”,“สถานะ”,“ID”]; function doPost(e) { try { const data = JSON.parse(e.postData.contents); const ss = SpreadsheetApp.getActiveSpreadsheet(); let sheet = ss.getSheetByName(SHEET_NAME); if (!sheet) { sheet = ss.insertSheet(SHEET_NAME); sheet.appendRow(HEADERS); sheet.getRange(1,1,1,HEADERS.length).setFontWeight(“bold”).setBackground(”#f97316”).setFontColor(”#ffffff”); sheet.setFrozenRows(1); } const lastRow = sheet.getLastRow(); const idCol = HEADERS.length; // คอลัมน์ ID if (data.action === “append”) { sheet.appendRow(data.row); } else if (data.action === “update”) { // หาแถวที่มี ID ตรงกัน const ids = sheet.getRange(2, idCol, Math.max(lastRow-1,1), 1).getValues().flat(); const idx = ids.indexOf(data.id); if (idx >= 0) sheet.getRange(idx+2, 1, 1, data.row.length).setValues([data.row]); } else if (data.action === “delete”) { const ids = sheet.getRange(2, idCol, Math.max(lastRow-1,1), 1).getValues().flat(); const idx = ids.indexOf(data.id); if (idx >= 0) sheet.deleteRow(idx+2); } return ContentService.createTextOutput(JSON.stringify({ok:true})).setMimeType(ContentService.MimeType.JSON); } catch(err) { return ContentService.createTextOutput(JSON.stringify({ok:false,error:err.message})).setMimeType(ContentService.MimeType.JSON); } }
5
วาง Web App URL ในไฟล์นี้
แทนที่ค่าใน SHEETS_WEBAPP_URL ด้วย URL ที่ได้จาก Deploy
const SHEETS_WEBAPP_URL = "https://script.google.com/macros/s/AKfy.../exec";
6
แชร์ไฟล์ HTML ให้ทีม
ทุกคนเปิดไฟล์ HTML เดียวกันในเบราว์เซอร์ → ข้อมูลจะ sync กันแบบ real-time ผ่าน Firebase และส่งไป Google Sheets ทันทีทุกครั้งที่บันทึก ✅
